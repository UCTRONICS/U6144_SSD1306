# U6143_ssd1306
## Enable i2c 
```bash
sudo raspi-config
```
Choose Interface Options 
Enable i2c

##  Download library 
```bash
git clone https://github.com/UCTRONICS/U6143_ssd1306.git
```
## Compile the source code 
```bash
cd U6143_ssd1306/C 
```
```bash
make 
```


## Run the display

```bash 
cd U6143_ssd1306/C 
```
```bash 
./display 
```
